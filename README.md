# 🏋️‍♂️ AI FitForAll – Smart Fitness & Diet Assistant

An intelligent and interactive web application that works as a **personalized AI Fitness Coach**.
Built with **Streamlit** and powered by **Groq AI**, this tool generates customized **workout routines, diet plans, and health metrics** based on user profiles.

---

## 📝 GitHub Profile Summary

**Project Title:** AI FitForAll
**Core Model:** Groq AI (LLM API)
**Short Bio:** An end-to-end AI fitness coach application that converts user health data into personalized workout and Indian diet plans using BMI, BMR, and TDEE calculations. It demonstrates skills in **Python, Streamlit, AI API integration, and health data processing**.

---

## 🧠 About the Project

AI FitForAll is a smart fitness web application that generates **personalized workout and diet plans** based on **BMI, BMR, and TDEE calculations**.

It works like a **virtual AI fitness coach** that understands user needs and provides **simple, affordable, and accessible health guidance**.

🎯 **Goal:**
To provide affordable and accessible fitness guidance for **students, adults, and elders** through AI-powered automation.

---

## 🚀 Features

* 💪 Personalized Workout Plan (7-day schedule)
* 🥗 Personalized Diet Plan (Indian diet supported 🇮🇳)
* 🤖 AI Fitness Coach (Groq AI powered chatbot)
* 📊 BMI, BMR, and TDEE Calculation
* 📈 Progress Tracking
* 📄 PDF Export for Workout & Diet Plan
* 🌐 Simple and Clean Streamlit Web Interface
* 🧑‍🤝‍🧑 Works for all age groups

---

## 🛠️ Tech Stack

**Frontend:** Streamlit
**Backend:** Python
**AI Model:** Groq API
**Libraries:** Pandas, NumPy, Matplotlib, FPDF/ReportLab
**UI:** HTML & CSS (Streamlit Components)
