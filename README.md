🧠 About the Project

AI FitForAll is a smart fitness web application that generates personalized workout and diet plans based on BMI, BMR, and TDEE calculations.
It acts like a virtual AI fitness coach that understands user needs and provides health guidance in a simple and interactive way.

The goal of this project is to provide affordable and accessible fitness guidance for everyone — students, adults, and elders.

✨ Features
💪 Personalized Workout Plan (7-day schedule)
🥗 Personalized Diet Plan (Indian diet supported 🇮🇳)
🤖 AI Fitness Coach (Groq AI powered chatbot)
📊 BMI, BMR, and TDEE Calculation
📈 Progress Tracking
📄 PDF Export for Workout & Diet Plan
🌐 Simple Streamlit Web Interface
🧑‍🤝‍🧑 Works for all age groups
🛠️ Tech Stack
🐍 Python
🌐 Streamlit
🤖 Groq AI API
📊 BMI / BMR / TDEE Logic
📄 PDF Generator
🎨 HTML & CSS (Streamlit UI)
